A convenção que usaremos para
criar nosso programa é adotar
letras minúsculas e não usar
acentos e, claro, usar a extensão .html.

Outro ponto importante é que
toda alteração feita no
arquivo .html deve ser salva
e, além disso, o navegador
precisa recarregar a página
para que a última alteração do
arquivo entre em vigor.
